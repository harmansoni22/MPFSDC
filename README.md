# MPFSDC - Madhya Pradesh Forest State Development Corporation

A modern, responsive static website for the Madhya Pradesh Forest State Development Corporation, built with HTML5, CSS3, and JavaScript.

## Features

- 🌱 **Responsive Design**: Fully responsive layout that works on all devices
- 🎨 **Modern UI**: Clean, professional design with Tailwind CSS
- ⚡ **Interactive Elements**: Smooth animations, hover effects, and transitions
- 🔍 **Search Functionality**: Built-in search modal for content discovery
- 📱 **Mobile-Friendly**: Optimized mobile navigation and touch interactions
- ♿ **Accessibility**: WCAG compliant with keyboard navigation and screen reader support
- 🖼️ **Gallery**: Interactive photo gallery with lightbox functionality
- 📋 **Contact Form**: Functional contact form with validation and feedback
- 📊 **Dynamic Content**: Animated counters and statistics

## Pages & Sections

- **Home**: Hero section with call-to-action and quick links
- **About Us**: Company overview, leadership information, and key statistics
- **Services**: Comprehensive list of forest management services
- **Projects**: Ongoing and completed projects with details
- **Auctions**: Current and upcoming auction schedules
- **Gallery**: Photo gallery showcasing forest activities
- **Contact**: Contact information and inquiry form

## Technology Stack

- **HTML5**: Semantic markup for structure
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Vanilla JavaScript**: Interactive functionality and animations
- **Font Awesome**: Icon library for UI elements
- **Intersection Observer API**: For scroll animations

## Getting Started

### Prerequisites

- A modern web browser
- Local web server (optional but recommended)

### Installation

1. Clone or download the project files
2. Navigate to the project directory
3. Start a local web server:

```bash
# Using Python
python -m http.server 8080

# Using Node.js (if you have http-server installed)
npx http-server -p 8080

# Using PHP
php -S localhost:8080
```

4. Open your browser and navigate to `http://localhost:8080`

## Customization

### Colors
The primary color scheme uses green tones to represent the forest theme. Colors can be easily customized by modifying the Tailwind CSS classes in the HTML file.

### Content
All text content is stored directly in the HTML file for easy editing. Images can be added to the `assets/` directory and referenced accordingly.

### Animations
Animation speeds and effects can be adjusted in the `styles.css` file by modifying the animation keyframes and transition properties.
